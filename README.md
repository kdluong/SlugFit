# Slugmeet - Find a time for UCSC Slugs to meet

A simple cross platform (IOS and Android) when to meet app made with React Native.

## Installation

Install Expo ?

## Run

- `yarn ios` to run IOS simulator
- `yarn run android` to run Android simulator
